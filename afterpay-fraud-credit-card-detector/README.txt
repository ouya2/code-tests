# About
A fraud credit card number detector.

# Requirement
If the project is imported in intellij IDE, lombok plugin for IDE is required.

# Sample
Expected input format:
10d7ce2f43e35fa57d1bbf8b1e2, 2014-04-29T13:15:54, 10.00
67r5c6ef43e35fa57d1byu87kk4, 2014-06-29T13:15:54, 20.00

# Note
If the data value is missing in a transaction, the transaction will be ignored.
